searchlog.sample.txt: 
A search engine log

data format:
[User visit time]\t[User ID]\t[Search keyword]\t[Rank of this URL in search results]\t[Click order by user]\t[link to URL]

example:
00:00:18	14376974823313943	[www.yahoo.com]	1 4	www.yahoo.com/

Note that [Rank of this URL in search results] and [Click order by user] are separated by a single space, and all other terms are separated by a tab space.
